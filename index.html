PROJECTS

Main Project

CLEANER ROBOT: My Main Project, Cleaner Robot is a six wheeled structure created on the basis of the Rockerbogie mechanism put forward by NASA. Automatic obstacle detection is done in the robot with ultrasonic sensor(HCSR04) and and the microcontroller- Arduino-Uno. We have used six wheels and the 4-channel relay module is used th drive the motor in the pattern of motion required as per the program.

Mini Project

DRIENESS INDICATION SYSTEM: My Mini Project, Automatic Drieness indication System works with the microcontroller Arduino-Uno and sensor used is Load cell module. When The object dries, the water content will go down, similar is the overall weight of the object. This loss of water is identified using the sensor and the microcontroller with the help of the amplifier module(HX711) and indication is produced.
